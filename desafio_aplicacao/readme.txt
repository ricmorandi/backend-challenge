Olá!

Acabei desenvolvendo um pouco mais do que foi solicitado,mas as funções solicitadas estão prontas e é possível enviar e
receber dados da API.

Tudo foi desenvolvido tendo como base um localhost, logo, os arquivos devem ser colocados na pasta do servidor.

Ao acessar a página, o arquivo index.html irá guiar a navegação com tranquilidade.

É importante antes de iniciar qualquer processo de cadastro, rodar o script de criação da base de dados que se encontra
nessa pasta (createDB.sql). Foi utilizado um banco de dados MySQL.
